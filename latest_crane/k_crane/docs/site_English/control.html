Control

This package is mainly a program for operating gazebo from within the program. It also includes a PID parameter setting file for each joint.

├── ds4toangle.cpp : Program to move crane from PS4 controller
├── go_initial_pose.cpp : Program to move to desired initial position
├── initial_joint12.cpp : Program to set joint12 of rope to 0 degree
├── initial_joint23.cpp : Program to set joint 23 of rope to 0 degree
├── initial_joint4.cpp : A program to make the joint 4 of the rope joint 0 degree
├── initial_joint_boom.cpp : Initial position movement program for testing
├── initial_joint_boom_pi.cpp : Initial position movement program for testing
├── initial_state.cpp : Initial position movement program for testing
├── integral.cpp : Program that integrates velocity trajectories and converts them into position trajectories
├── joint_trajectory_action.cpp : A program that executes a given trajectory (speed, position, time)
├── logger.cpp : Program for logging
├── move_trapezoid.cpp : For test, trapezoidal acceleration program
├── pause_gazebo.cpp : Program to suspend gazebo
├── sano_joint_trajectory_action.cpp : A program to send Sano-sensei's vibration control trajectory
└── unpause_gazebo.cpp : Program to resume gazebo from pause

If you want to move the crane from moment to moment, you can do so by publishing to the /k_crane/k_crane_joint_controller/command topic. See k_crane_control/src/ds4toangle.cpp for details.
