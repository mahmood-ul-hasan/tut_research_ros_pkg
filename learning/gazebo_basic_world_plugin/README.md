### **Installation** 
Download this package in the _src_ folder of your ROS workspace

```git clone https://github.com/jocacace/gazebo_basic_world_plugin```

Compile the package

```cd gazebo_basic_world_plugin```  
```mkdir build && cd build```  
```cmake ..```  
```make```
